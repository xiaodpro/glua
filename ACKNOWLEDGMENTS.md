# Acknowledgments

## 致谢

### 项目背景

This project (glua) is based on the open-source project [gopher-lua](https://github.com/yuin/gopher-lua), which was created by Yusuke Inuzuka. We would like to express our sincere gratitude to the original author and all contributors of gopher-lua for their excellent work.

本项目（glua）基于开源项目 [gopher-lua](https://github.com/yuin/gopher-lua) 改造而成，该项目由 Yusuke Inuzuka 创建。我们诚挚地感谢 gopher-lua 的原始作者和所有贡献者的出色工作。

### 主要改动

In order to differentiate from the original project, we have renamed the module from `github.com/yuin/gopher-lua` to `glua`. This change was made to avoid conflicts with the official gopher-lua library and to clearly indicate that this is a modified version.

为了区别于原项目，我们将模块从 `github.com/yuin/gopher-lua` 更名为 `github.com/xiaodpro/glua`。这一改动是为了避免与官方 gopher-lua 库发生冲突，并明确表明这是一个修改版本。

### 使用建议

If you need a Lua runtime environment that fully supports the official Lua standards, we recommend using the original [gopher-lua](https://github.com/yuin/gopher-lua) library. This modified version (glua) may have specific adjustments and enhancements that are tailored to our specific use cases.

如果您需要一个完全支持官方 Lua 标准的 Lua 运行环境，我们建议使用原始的 [gopher-lua](https://github.com/yuin/gopher-lua) 库。这个修改版本（glua）可能包含特定于我们用例的调整和增强。
